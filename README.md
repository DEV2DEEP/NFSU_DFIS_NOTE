M.Sc. Digital Forensics & Information Security Notes

📚 About This Repository

This repository contains complete notes, study materials, practicals, assignments, lab manuals, reference materials, and important resources for the M.Sc. Digital Forensics & Information Security (DFIS) course offered by National Forensic Sciences University (NFSU).

The notes are prepared semester-wise and subject-wise for easy understanding and examination preparation. The repository is useful for:

🎓 M.Sc. DFIS Students
🔐 Cyber Security Enthusiasts
🕵️ Digital Forensics Learners
💻 Ethical Hackers & Security Researchers
📖 Students preparing for university examinations and practicals

The syllabus and course structure are based on the official NFSU curriculum.

🏛 University Information

University: National Forensic Sciences University (NFSU)
Department: School of Cyber Security & Digital Forensics
Course: M.Sc. Digital Forensics & Information Security (DFIS)
Program Type: Postgraduate Degree Program
